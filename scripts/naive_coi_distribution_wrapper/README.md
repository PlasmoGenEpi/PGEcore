# Naive COI Distribution Estimation

Contents: 
* [Tool Information](#tool-information)

## Tool Information

This tool provides a naive summary of the distribution of COI calls across samples.

## Script Usage 
```
Rscript naive_coi_distribution_wrapper.R \
  --coi_calls data/example_coi_table.tsv \
  --output coi_distribution.tsv
```
